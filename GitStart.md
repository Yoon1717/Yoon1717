# GitStart
## Markdown
1. **헤더(Headers)**
```
    백틱(`) <-- 3개 코드라인 markdown 문법이 적용안된다
  # 헤더 1
 ## 헤더 2
### 헤더 3
```
## 헤더 2
#### 헤더 4

2. 굵게
``` **굵게** ```
**굵게**

3. 기울임
``` *기울임* ```
*기울임*

4. 링크
```[링크텍스트](URL)```
[링크텍스트](URL)

5. 이미지
```![대체 텍스트](이미지_URL)```
![대체 텍스트](이미지_URL)

6. 리스트
``` 1. 번째 2. 번째 ```
  1. 번째
  2. 번째

7. 인용
``` > 인용문 ```
> 인용문
> 
8. 1줄 코드
``` `코드` ```
`int i = 0; `

9. 구분선
